Ava's BookStore

Ava's BookStore was my capstone project at Texas State Technical College.

This windows application is a fast and easy way to navigate through an assortment of books and magazines at your leisure. You will be able to make quick and easy purchases and apply discounts to your order in a simple manner with no hassle.

Ava's Bookstore is my first completed and fully tested Windows Application that I've developed.

To view all features and see what the capability of the application is, please view my video above.

With this application you will be able to access all the books/magazines without having to login and see what is available. A simple click and add to cart will make it easy and not a hassle to make a quick purchase at your leisure.

Managers/Employees also have a completely different section of the application that they can use to add, remove and update data as they please. This includes add/removing/updating inventory, customers, managers, and discounts. They also have permission to print all inventory reports as well as Sales Reports with different filters such as daily, weekly, and monthly sales.

This application was developed in Visual Studios as a Windows Form Application with C#. It also is fully capable of accessing and updating a database as needed. The SQL Server Type/Dialect used for this application was Microsoft SQL. The target enviornment is Windows 10 and the development environment was Visual Studio 2019 Community Edition.
